To install and set up the project, follow these steps:

For Backend :
1. Clone the project repository from repository URL.
2. run command "cd Backend"
3. then run "npm i "
4. and after than 'npm start'

For Frontend:
1. run "cd Backend"
2. then run "npm i"
3. and after that "npm start"
